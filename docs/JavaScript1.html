JavaScript 是什么？JavaScript 简介

一、定义与起源

JavaScript（简称 JS）是一种基于原型的、弱类型的、动态语言，主要用于浏览器端脚本编程，也能在服务端运行（如 Node.js）。

  • 起源：1995 年由 Netscape 公司的 Brendan Eich 设计，最初名为 LiveScript。
  • 为了配合 Java 浪潮，改名为 JavaScript。
  • JavaScript 的语言规范由 ECMA 国际标准化组织负责，标准名称为 ECMAScript（ES）。

二、主要特点

  • 解释执行：代码无需编译即可运行。
  • 动态类型：变量类型在运行时决定。
  • 基于原型：对象继承基于原型链。
  • 单线程：通过事件循环机制处理异步。
  • 支持多种范式：命令式、面向对象、函数式。
  • 跨平台：现代浏览器均支持 JS 运行。

三、核心组成部分

  1. ECMAScript：语言本体（语法、类型、操作符等）
  1. DOM：文档对象模型，用于操作 HTML/XML
  1. BOM：浏览器对象模型，如 window、location、history 等

四、应用场景

  • 网页交互开发
  • Node.js 服务端开发
  • 移动端混合开发（如 React Native）
  • 桌面应用（如 Electron）
  • 脚本自动化、爬虫、工具链等

Node.js 是什么？Node.js 简介

一、定义与背景

Node.js 是一个基于 Google Chrome 的 V8 JavaScript 引擎构建的 JavaScript 运行环境，可用于服务端开发。它由 Ryan Dahl 于 2009 年发布。

二、核心特性

  • 非阻塞 I/O：适合高并发应用。
  • 单线程事件循环：避免传统多线程带来的资源开销。
  • 跨平台：支持 Linux / Windows / macOS。
  • npm：Node.js 的包管理工具，是世界上最大的开源生态系统之一。

三、应用场景

  • RESTful API 后端服务
  • SSR 服务端渲染（如 Next.js）
  • 实时通信系统（如聊天室、WebSocket）
  • 命令行工具、构建工具

四、Node 与 JS 的关系

Node.js 使用 JavaScript 编程语言，但运行在服务器端，不具有浏览器环境中的 BOM 和 DOM。Node 提供自己的 API（如 fs、http、events）。

javascript

三、执行顺序说明

  • HTML 会从上到下顺序加载。
  • <script> 标签执行 JS 时会暂停 HTML 渲染。
  • 外部 script 默认是同步执行，可加 defer / async 改变行为。

JavaScript 中的几个重要概念

本文讲解了 JavaScript 中的几个简单概念，包括标识符、关键字、保留字、大小写和字面量。这些基本概念虽然不能直接提升我们的编程能力，但它们是 JavaScript 的基本组成元素。

1. 标识符

所谓标识符（Identifier），就是名字。JavaScript 中的标识符包括变量名、函数名、参数名、属性名、类名等。

合法的标识符应遵循以下强制规则：

  • 第一个字符必须是字母、下划线（_）或美元符号（$）。
  • 除了第一个字符外，其他位置可以使用 Unicode 字符。一般建议仅使用 ASCII 编码的字母，不建议使用双字节的字符。
  • 不能与 JavaScript 关键字、保留字重名。
  • 可以使用 Unicode 转义序列。例如，字符 a 可以使用 \u0061 表示。

示例

在下面示例中，str 就是变量的名字：

var str = "CSDN博客：http://www.csdn.net/";
document.write(str);

第 1 行代码定义了一个变量，名字为 str，第 2 行通过 str 这个名字使用了变量。

2. 关键字

关键字（Keyword）是 JavaScript 语言内部使用的一组名字（或称为命令）。这些名字具有特定的用途，用户不能自定义同名的标识符。

JavaScript 关键字包括：

break      delete     if         this       while
case       do         in         throw      with
catch      else       instanceof try
continue   finally    new        typeof
debugger   for        return     var
default    function   switch     void

3. 保留字

保留字是 JavaScript 语言内部预备使用的一组名字（或称为命令）。这些名字目前还没有具体的用途，是为 JavaScript 升级版本预留备用的，建议用户不要使用。

JavaScript 保留字包括：

abstract    double       goto         native       static
boolean     enum         implements   package      super
byte        export       import       private      synchronized
char        extends      int          protected    throws
class       final        interface    public       transient
const       float        long         short        volatile

ECMAScript 3 将 Java 所有关键字都列为保留字，而 ECMAScript 5 规定较为灵活，例如：

  • 在非严格模式下，仅规定 class、const、enum、export、extends、import、super 为保留字，其他 ECMAScript 3 保留字可以自由使用；
  • 在严格模式下，ECMAScript 5 变得更加谨慎，严格限制 implements、interface、let、package、private、protected、public、static、yield、eval（非保留字）、arguments（非保留字）的使用。

JavaScript 预定义了很多全局变量和函数，用户也应该避免使用它们，具体说明如表所示。

JavaScript 预定义全局变量和函数：

arguments    encodeURI       Infinity      Number        RegExp
Array        encodeURIComponent  isFinite  Object        String
Boolean      Error           isNaN         parseFloat    SyntaxError
Date         eval            JSON          parseInt      TypeError
decodeURI    EvalError       Math          RangeError    undefined
decodeURIComponent  Function  NaN          ReferenceError URIError

不同的 JavaScript 运行环境都会预定义一些全局变量和函数，上表列出的仅针对 Web 浏览器运行环境。

无论是在严格模式下还是在非严格模式下，都不要在定义变量名、函数名或者属性名时使用上面列举出的保留字，以免同学们入坑。

4. 区分大小写

JavaScript 严格区分大小写，所以 Hello 和 hello 是两个不同的标识符。

为了避免输入混乱和语法错误，建议采用小写字符编写代码，在以下特殊情况下可以使用大写形式：

  • 构造函数的首字母建议大写。构造函数不同于普通函数。

示例

下面示例调用预定义的构造函数 Date()，创建一个时间对象，然后把时间对象转换为字符串显示出来。

var d = new Date();  // 获取当前日期和时间
document.write(d.toString());  // 显示日期

如果标识符由多个单词组成，可以考虑使用骆驼命名法——除首个单词外，后面单词的首字母大写。例如：

typeOf();
printEmployeePaychecks();

提示：上述都是约定俗成的一般习惯，不构成强制性要求，用户可以根据个人习惯进行命名。

5. 字面量

字面量（Literal）也叫直接量，就是具体的值，即能够直接参与运算或显示的值，如字符串、数值、布尔值、正则表达式、对象直接量、数组直接量、函数直接量等。

示例

下面示例分别定义不同类型的直接量：字符串、数值、布尔值、正则表达式、特殊值、对象、数组和函数。

""            // 空字符串直接量
1             // 数值直接量
true          // 布尔值直接量
/a/g          // 正则表达式直接量
null          // 特殊值直接量
{}            // 空对象直接量
[]            // 空数组直接量
function(){}  // 空函数直接量，也就是函数表达式

JS 注释（多行注释 + 单行注释）

JavaScript 中的注释用于提高代码的可读性和可维护性。注释不会被浏览器解析执行。

1. 单行注释

单行注释以 // 开头，// 后面的内容直到该行结束都被视为注释.

示例：

// 程序描述
function toStr(a) {  // 块描述
  // 代码段描述
  return a.toString();  // 语句描述
}

在使用单行注释时，// 后面的同一行内的任何字符或代码都会被忽略，不再解析。

2. 多行注释

多行注释以 /* 开始，以 */ 结束，/* 和 */ 之间的所有内容都被视为注释。

示例：

/*
 * jQuery JavaScript Library v3.3.1
 * https://jquery.com/
 * Includes Sizzle.js
 * https://sizzlejs.com/
 * Copyright JS Foundation and other contributors
 * Released under the MIT license
 * https://jquery.org/license
 * Date: 2019-08-21 T 17:24 Z
 */

在多行注释中，包含在 /* 和 */ 符号之间的任何字符都被视为注释文本而忽略掉。

3. 使用注释来阻止程序执行

在调试过程中，可以使用注释来临时禁用某段代码，而不是直接删除

示例：

// function oldFunction() {
//   // ...
// }

这样做可以方便地在需要时恢复代码，同时不会影响程序的其他部分。

4. HTML 注释

在 HTML 文件中，JavaScript 代码通常放在 <script> 标签中。虽然 HTML 支持 <!-- --> 作为注释，但在 JavaScript 中不推荐使用这种方式。

示例（不推荐）：

<script>
<!--
  // JavaScript 代码
-->
</script>

建议在 JavaScript 中使用 // 或 /* */ 来添加注释，以保持代码的一致性和可读性。

JavaScript 变量定义和赋值（结构化整理）

一、变量的命名规则

变量名必须遵守 JavaScript 的标识符规范：

  • ✅ 可包含字母、数字、下划线（_）、美元符号（$）
  • ❌ 不可包含空格、汉字
  • ❌ 不可以数字开头
  • ❌ 不可与关键字或保留字重名
  • ✅ 建议使用有意义的命名（如 userName, totalCount）
  • ✅ 多词建议使用小驼峰命名法（如 fileName, dataList）

二、定义变量的方式（var / let / const）

关键字引入版本特点说明
varES5函数作用域，允许重复声明，存在变量提升
letES6块级作用域，不允许重复声明，无变量提升
constES6块级作用域，必须初始化，值不可更改（常量）

示例：

var a;       // 声明变量 a
let b;       // 块作用域变量
const PI = 3.14;  // 声明常量，必须立即赋值

三、变量赋值

变量定义后，可以通过 = 为其赋值：

var name = "小明";        // 定义并赋值
let age = 18;
const country = "中国";   // 一定要赋初值

也可先声明后赋值：

let x;
x = 5;

支持同时声明多个变量：

var a = 1, b = 2, c = 3;

四、变量提升（Hoisting）

使用 var 声明的变量会被“提升”到当前作用域顶部，但不会提升赋值。


console.log(x); // 输出 undefined（已声明但未赋值）
var x = 10;

使用 let / const 的变量不会被提升，访问会抛出错误：

console.log(y); // ReferenceError
let y = 5;

五、let 与 const 的细节区别

let：

  • 可以声明后赋值
  • 不允许重复声明
  • 有块级作用域限制

const：

  • 声明时必须赋值
  • 值不可变（对对象/数组引用不变，但属性可变）
  • 不可重复声明
const person = { name: "Tom" };
person.name = "Jerry"; // ✅ 可以修改属性
person = {}; // ❌ 报错：不能重新赋值

JavaScript 数据类型（基本类型 + 引用类型）

在 JavaScript 中，变量可以存储不同类型的数据。JavaScript 是动态类型语言，变量的数据类型可以在运行时改变。

一、typeof 操作符

用于检查变量的数据类型：

typeof "hello"      // "string"
typeof 123          // "number"
typeof true         // "boolean"
typeof undefined    // "undefined"
typeof null         // "object"（注意：这是 JavaScript 的历史遗留问题）
typeof {}           // "object"
typeof []           // "object"
typeof function(){} // "function"

二、基本数据类型（Primitive Types）

  1. String（字符串）
    let str = "hello";
let str2 = 'world';
let str3 = "He said \"Hi\""; // 使用转义字符
  1. Number（数字）
    • 支持整数和浮点数，不区分 int 和 float。
    • 支持科学记数法（如 123e5）
    • 特殊值：Infinity、Infinity、NaN
    let a = 123;
let b = 3.14;
let c = 123e5;     // 12300000
let d = 123e-5;    // 0.00123
let e = 0 / 0;     // NaN
  1. Boolean（布尔）
    let isTrue = true;
let isFalse = false;
  1. Null（空）
    • 表示“无”或“空对象值”。
    • typeof null 返回 "object"（设计缺陷）。
    let empty = null;
  1. Undefined（未定义）
    • 声明了变量但未赋值，其值为 undefined。
    let x;
console.log(x); // undefined
  1. Symbol（符号）
    • ES6 引入，用于创建唯一标识符。
    let sym1 = Symbol("id");
let sym2 = Symbol("id");
console.log(sym1 === sym2); // false

三、引用数据类型（Reference Types）

引用类型的变量保存的是“对象的引用地址”。

  1. Object（对象）
    let person = {
  name: "Tom",
  age: 30
};
console.log(person.name); // Tom
  1. Array（数组）
    • 有序集合，支持多种数据混用
    let arr = [1, "hello", true, null];
console.log(arr[1]); // hello
  1. Function（函数）
    • 一等公民，可赋值给变量、传参、返回
    function greet(name) {
  return "Hello " + name;
}

let sayHi = function(name) {
  return "Hi " + name;
};

let msg = greet("Alice");

JavaScript 输出语句汇总（5 种）

在 JavaScript 中，有多种方式将信息输出到用户界面或开发者控制台。以下是最常用的 5 种输出语句及其特点：

1. alert() 弹窗提示

  • 用于显示一个消息框，常用于调试或提醒用户。
  • 会阻塞后续 JS 执行，直到用户关闭提示框。
alert("欢迎来到我的网站！");

2. confirm() 确认框

  • 弹出一个带“确定”和“取消”的对话框。
  • 返回 true 表示点击“确定”，返回 false 表示点击“取消”。
let result = confirm("你确定要删除吗？");
if (result) {
  alert("已删除！");
}

3. console.log() 控制台输出（推荐用于调试）

  • 将信息输出到浏览器开发者工具中的控制台。
  • 不会中断程序运行，是开发过程中最常用的调试方式。
let a = 10;
console.log("变量 a 的值是：" + a);

4. document.write() 页面直接输出内容

  • 将信息直接写入到 HTML 文档中。
  • 如果在文档加载完成后使用，会清除页面已有内容。
document.write("<h1>Hello World!</h1>");

⚠️ 注意： 不推荐在现代项目中使用，除非在教学或 HTML 初学阶段。

5. innerHTML 修改页面元素内容

  • 用于设置或读取 HTML 元素的内容。
  • 更加灵活，可将变量、标签、结构输出到页面。

HTML 结构：

<p id="demo"></p>

JS 输出：

document.getElementById("demo").innerHTML = "Hello JavaScript!";

JavaScript 条件判断语句（if / else）

在 JavaScript 中，可以使用 if、else if、else 语句根据不同条件执行不同代码，从而实现基本的程序逻辑控制。

一、if 语句

语法：

if (条件) {
  // 条件为 true 时执行的代码
}

示例：

let score = 85;

if (score >= 60) {
  console.log("及格");
}

二、if...else 语句

用于根据条件执行两个不同分支的代码。

语法：

if (条件) {
  // 条件为 true
} else {
  // 条件为 false
}

示例：

let isLogin = false;

if (isLogin) {
  console.log("欢迎回来！");
} else {
  console.log("请先登录！");
}

三、if...else if...else 多条件语句

用于处理多个判断条件。

语法：

if (条件1) {
  // 条件1为 true
} else if (条件2) {
  // 条件2为 true
} else {
  // 以上条件都不满足
}

示例：

let score = 76;

if (score >= 90) {
  console.log("优秀");
} else if (score >= 75) {
  console.log("良好");
} else if (score >= 60) {
  console.log("及格");
} else {
  console.log("不及格");
}

四、嵌套 if 语句（不推荐嵌套过深）

let userType = "admin";
let isOnline = true;

if (userType === "admin") {
  if (isOnline) {
    console.log("管理员在线");
  } else {
    console.log("管理员不在线");
  }
}

五、逻辑运算符搭配 if 使用

let age = 20;
let hasID = true;

if (age >= 18 && hasID) {
  console.log("可以进入");
} else {
  console.log("禁止进入");
}

小技巧：简化 if 判断

可以结合三元运算符：

let isVIP = true;
let message = isVIP ? "欢迎 VIP 用户" : "欢迎普通用户";
console.log(message)

JavaScript switch / case 条件判断语句

当存在多个固定值判断的情况，使用 switch / case 比 if / else if 更清晰简洁。

一、基本语法

switch (表达式) {
  case 值1:
    // 表达式等于值1时执行
    break;
  case 值2:
    // 表达式等于值2时执行
    break;
  ...
  default:
    // 所有 case 不匹配时执行（可选）
}

二、示例：星期转换

let day = 3;
switch (day) {
  case 1:
    console.log("星期一");
    break;
  case 2:
    console.log("星期二");
    break;
  case 3:
    console.log("星期三");
    break;
  default:
    console.log("不是 1~3 之间的数字");
}

✅ break 是必须加的，否则会“贯穿执行”后面所有 case。

三、没有 break 的情况（fall-through）

let color = "red";

switch (color) {
  case "red":
    console.log("红色");
  case "blue":
    console.log("蓝色");
  default:
    console.log("未知颜色");
}

🔴 输出：

红色
蓝色
未知颜色

➡️ 因为没有 break，所以会“贯穿执行”下方所有语句。

四、default 子句（默认分支）

  • default 是可选的；
  • 通常放在最后（但也可以放前面，效果一样）；
  • 相当于 else 的作用，用来处理未匹配情况。

五、表达式是值匹配（不能写布尔判断）


let x = 10;

switch (x > 5) {
  case true:
    console.log("大于 5");
    break;
  case false:
    console.log("小于等于 5");
    break;
}

📝 虽然可以这样写，但不推荐，switch 更适用于多个值匹配的场景，而不是条件表达式。

六、多个 case 合并

多个 case 可合并逻辑：

javascript

let fruit = "香蕉";

switch (fruit) {
  case "苹果":
  case "香蕉":
  case "橘子":
    console.log("这是水果");
    break;
  case "白菜":
    console.log("这是蔬菜");
    break;
  default:
    console.log("未知类型");
}

JavaScript 循环语句汇总（while / do while / for）

循环语句用于重复执行代码块，直到指定条件不再满足。JavaScript 中主要提供三种基本循环结构：

一、while 循环

语法：

javascript
while (条件) {
  // 条件为 true 时重复执行的代码
}

示例：

javascript

let i = 0;
while (i < 5) {
  console.log(i);
  i++;
}

✔️ 特点：

  • 先判断条件，再执行循环体；
  • 适用于循环次数不确定的场景。

二、do...while 循环

语法：


do {
  // 至少执行一次
} while (条件);

示例：


let i = 0;
do {
  console.log(i);
  i++;
} while (i < 5);

✔️ 特点：

  • 先执行一次，再判断条件；
  • 用于必须执行一次以上的逻辑。

三、for 循环

语法：

javascript
for (初始化; 条件; 步进) {
  // 循环体
}

示例：

javascript

for (let i = 0; i < 5; i++) {
  console.log(i);
}

✔️ 特点：

  • 循环结构最清晰；
  • 适合循环次数确定的场景。

四、for 循环嵌套

可以在一个 for 循环中再嵌套另一个 for 循环：

javascript

for (let i = 1; i <= 3; i++) {
  for (let j = 1; j <= 3; j++) {
    console.log(`i=${i}, j=${j}`);
  }
}

五、for...in 循环（用于对象）

  • 遍历对象的属性名（键）

let obj = { name: "Tom", age: 18 };

for (let key in obj) {
  console.log(key + ": " + obj[key]);
}

📌 注意：数组也能用 for...in，但不推荐，顺序不可保证。

六、for...of 循环（用于数组、类数组、可迭代对象）

  • 遍历数组的值

let arr = ["apple", "banana", "orange"];

for (let item of arr) {
  console.log(item);
}

📌 只能用于可迭代对象（如数组、字符串、Map、Set 等），不能用于普通对象。

七、while / do...while / for 对比总结

结构条件判断时间是否至少执行一次使用场景
while循环前否条件先判断、次数不确定
do...while循环后✅ 是至少执行一次
for循环前否次数明确

JavaScript 跳出循环语句（break / continue / label）

在循环中，有时需要“提前跳出”或“跳过当前一次”，这时可以使用 break 和 continue，必要时也可以配合 label 使用。

一、break：跳出整个循环

用于立即终止当前所在的循环结构（for、while、do while、switch）。

语法：

javascript

break;

示例：

javascript

for (let i = 1; i <= 5; i++) {
  if (i === 3) {
    break; // 遇到 3 时停止整个循环
  }
  console.log(i); // 输出 1、2
}

二、continue：跳过当前一次循环

用于跳过本次循环的剩余语句，直接进入下次循环判断。

语法：

javascript
continue;

示例：


for (let i = 1; i <= 5; i++) {
  if (i === 3) {
    continue; // 跳过当前 i=3
  }
  console.log(i); // 输出 1、2、4、5
}

三、label（标签）+ break/continue：控制多层嵌套循环

break 和 continue 默认只控制内层循环。如果想要跳出外层循环，需要用 label 结合使用。

语法：


labelName: for (...) {
  for (...) {
    if (条件) break labelName;
  }
}

示例：跳出外层循环

javascript

outerLoop: for (let i = 0; i < 3; i++) {
  for (let j = 0; j < 3; j++) {
    if (i === 1 && j === 1) {
      break outerLoop; // 跳出 outerLoop
    }
    console.log(`i=${i}, j=${j}`);
  }
}

📌 输出：


i=0, j=0
i=0, j=1
i=0, j=2
i=1, j=0

示例：跳过外层当前循环（continue + label）


rowLoop: for (let i = 0; i < 3; i++) {
  for (let j = 0; j < 3; j++) {
    if (j === 1) {
      continue rowLoop; // 跳过当前行，进入下一 i
    }
    console.log(`i=${i}, j=${j}`);
  }
}

📌 输出：


i=0, j=0
i=1, j=0
i=2, j=0

四、小结

语句功能描述
break跳出当前循环/switch 结构
continue跳过当前一次，进入下一轮循环
label用于命名循环，配合跳出或跳过嵌套循环
