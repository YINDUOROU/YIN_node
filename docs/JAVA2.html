一、Java 数据类型（Data Types）分类

Java 的数据类型分为两大类：

类型分类说明
基本数据类型（PrimitiveTypes）值类型直接存储值，占用固定内存
引用数据类型（Reference Types）引用类型存储对象的地址或引用

一、基本数据类型（Primitive Types）

一、Java 一共提供了 8种基本数据类型，如下表所示：

类型名称占用字节取值范围或说明
byte字节型1 字节-128 到 127
short短整型2 字节-32,768 到 32,767
int整型4 字节-2³¹ 到 2³¹-1（最常用）
long长整型8 字节更大的整数，末尾加 L
float单精度浮点型4 字节末尾加 f，如：3.14f
double双精度浮点型8 字节默认浮点类型
char字符型2 字节存储单个字符或 Unicode
boolean布尔型1 字节（逻辑）只取 true 或 false

示例代码

public class DataTypeExample {
    public static void main(String[] args) {
        byte b = 100;
        short s = 30000;
        int i = 123456;
        long l = 9876543210L;

        float f = 3.14f;
        double d = 2.71828;

        char c = 'A';
        boolean flag = true;

        System.out.println("byte: " + b);
        System.out.println("short: " + s);
        System.out.println("int: " + i);
        System.out.println("long: " + l);
        System.out.println("float: " + f);
        System.out.println("double: " + d);
        System.out.println("char: " + c);
        System.out.println("boolean: " + flag);
    }
}

二、类型转换（Type Conversion）

🔹 自动类型转换（小 → 大）

int i = 100;
long l = i;         // int 自动转 long
float f = l;        // long 自动转 float

🔹 强制类型转换（大 → 小）

double d = 3.14;
int i = (int) d;    // 会丢失小数部分，结果是 3

三、字符与 ASCII / Unicode

char ch = 'A';      // 字符类型
int ascii = ch;     // 自动转成对应的 ASCII 值（65）
System.out.println(ascii);  // 输出 65

你还可以用 Unicode：

char ch2 = '\u4F60'; // Unicode 编码表示 "你"
System.out.println(ch2);   // 输出：你

  • 📘 中文ASCII码表参考：https://tool.oschina.net/commons?type=4
  • 📘 Unicode 中文手册版：https://zh.wikipedia.org/wiki/Unicode

📎Unicode Character Table

四、浮点型的特殊注意

  • float 必须加 f，如 3.14f
  • double 是默认浮点类型，不用加后缀
  • 精度问题要注意：0.1 + 0.2 != 0.3

五、boolean 只能取两值

boolean flag = true;
if (flag) {
    System.out.println("这是 true");
}

一、引用数据类型（Reference Types）

一、概览

类型说明
String字符串类型
数组一组相同类型元素的集合
类（Class）面向对象的核心，自定义数据结构
接口（Interface）一种行为规范
枚举（Enum）有限个常量值
对象（Object）类的实例
📌 所有引用类型的变量，保存的都是“对象的地址”，并不直接存储值。

二、String（字符串）

🔹 定义方式

String s1 = "Hello";  // 推荐方式，常量池复用
String s2 = new String("World"); // new 创建新对象

🔹 常用方法

s.length();           // 字符串长度
s.charAt(0);          // 取字符
s.equals(s2);         // 比较值是否相同
s.toUpperCase();      // 转大写
s.substring(1, 3);    // 截取子串
s.replace("a", "b");  // 替换字符
s.contains("lo");     // 是否包含

🔹 字符串是不可变对象（immutable）：修改内容实际上是创建新字符串。

三、数组（Array）

🔹 定义方式

int[] nums = new int[3];             // 长度为3，默认值0
String[] names = {"Tom", "Jerry"};   // 初始化数组

🔹 使用方式

nums[0] = 10;
System.out.println(nums.length);     // 数组长度

🔹 遍历数组

for (int i = 0; i < nums.length; i++) {
    System.out.println(nums[i]);
}

for (int n : nums) {  // 增强for循环
    System.out.println(n);
}

四、类与对象（Class & Object）

🔹 自定义类

class Person {
    String name;
    int age;

    void sayHello() {
        System.out.println("Hello, my name is " + name);
    }
}

🔹 创建对象

Person p = new Person();  // 创建实例
p.name = "Alice";
p.sayHello();             // 调用方法

五、引用类型的默认值

类型默认值
引用类型（如 String, Object, Array）null
基本类型（如 int, boolean）0 / false

六、引用类型与堆内存

  • 引用类型的数据存储在堆（Heap）内存中
  • 变量本身在栈（Stack）中存放引用（地址）
  • 比较两个对象的地址要用 ==，比较内容要用 .equals()
String a = new String("hello");
String b = new String("hello");
System.out.println(a == b);         // false（地址不同）
System.out.println(a.equals(b));    // true（内容相同）

二、变量与常量

一、变量（Variable）

🔹 定义格式

<数据类型> <变量名> = <初始值>;

🔹 示例：

int age = 25;
double height = 1.75;
String name = "Alice";

🔹 注意：

  • 变量名一般使用 小驼峰命名法（如：userName、maxValue）
  • 定义变量必须指定类型（Java 是强类型语言）

✅ Java 中变量的分类

分类方式类型说明
按定义位置局部变量在方法内部定义，只能在方法内使用
成员变量（实例变量）在类中定义，不加 static，属于对象
静态变量（类变量）使用 static 关键字，属于类
按数据类型基本类型变量如 int, boolean 等
引用类型变量如 String, 数组, 类 等

✅ 示例区分：

public class User {
    // 成员变量（实例变量）
    String name;

    // 静态变量（类变量）
    static int count = 0;

    public void show() {
        // 局部变量
        int age = 18;
        System.out.println(name + " - " + age);
    }
}

二、常量（Constant）

  • 一般使用 全大写字母 + 下划线 的形式命名，如：MAX_LENGTH、DEFAULT_TIMEOUT

🔹 示例：

public class Config {

三、运算符

一、Java 运算符总览

Java 中的运算符可以分为以下几类：

类别代表符号用途举例
算术运算符+ - * / % ++ --数值计算
赋值运算符= += -= *= /= %=给变量赋值
比较运算符== != > < >= <=判断两个值的关系
逻辑运算符`&&
位运算符`&^ ~ << >> >>>`
条件运算符? :三元条件表达式
instanceofinstanceof判断对象类型

二、算术运算符（Arithmetic）

int a = 10;
int b = 3;
System.out.println(a + b); // 13
System.out.println(a - b); // 7
System.out.println(a * b); // 30
System.out.println(a / b); // 3（整数除法）
System.out.println(a % b); // 1（取余数）

自增/自减运算符

int x = 5;
System.out.println(x++); // 5（先用后加）
System.out.println(++x); // 7（先加后用

三、赋值运算符（Assignment）

int a = 5;
a += 2; // 相当于 a = a + 2;
a *= 3; // 相当于 a = a * 3;

四、比较运算符（Relational）

int a = 10;
int b = 5;

System.out.println(a > b);   // true
System.out.println(a == b);  // false
System.out.println(a != b);  // true

五、逻辑运算符（Logical）

适用于 boolean 类型：

boolean a = true;
boolean b = false;

System.out.println(a && b);  // false（与：两者都为true才为true）
System.out.println(a || b);  // true（或：只要一个为true就为true）
System.out.println(!a);      // false（非）

六、位运算符（Bitwise）

（适合了解，有些算法题会考）

int a = 3;  // 011
int b = 5;  // 101

System.out.println(a & b); // 1（按位与）
System.out.println(a | b); // 7（按位或）
System.out.println(a ^ b); // 6（异或）
System.out.println(~a);    // -4（取反）

七、三元运算符（Ternary Operator）

int a = 10, b = 20;
int max = (a > b) ? a : b;
System.out.println(max); // 20

八、instanceof 运算符

String s = "Hello";
System.out.println(s instanceof String); // true

九、运算符优先级（简略版）

优先级从高到低（只列部分常用）：

  1. () 圆括号
  1. ++ --
  1. / %
  1. + -
  1. > < >= <=
  1. == !=
  1. &&
  1. ||
  1. = += -= *=

四、流程控制语句

一、条件判断语句（选择结构）

🔹 1. if 语句

int age = 20;

if (age >= 18) {
    System.out.println("成年人");
} else {
    System.out.println("未成年人");
}

🔹 2. 多重 if...else if...else

int score = 85;

if (score >= 90) {
    System.out.println("优秀");
} else if (score >= 70) {
    System.out.println("良好");
} else {
    System.out.println("需努力");
}

🔹 3. 嵌套 if

if (age > 0) {
    if (age < 18) {
        System.out.println("未成年人");
    }
}

✅ 二、switch 多分支语句

适用于整型、字符串、枚举等值判断。

int day = 2;

switch (day) {
    case 1:
        System.out.println("星期一");
        break;
    case 2:
        System.out.println("星期二");
        break;
    default:
        System.out.println("其他");
}

🔸 Java 7+ 可以 switch 字符串

String role = "admin";

switch (role) {
    case "admin":
        System.out.println("管理员");
        break;
    case "user":
        System.out.println("普通用户");
        break;
}

✅ 三、循环语句（for / while / do while）

🔹 1. for 循环

java
复制编辑
for (int i = 0; i < 5; i++) {
    System.out.println("i = " + i);
}

🔹 2. 增强 for-each 循环（用于遍历数组/集合）

java
复制编辑
int[] nums = {1, 2, 3};
for (int n : nums) {
    System.out.println(n);
}

🔹 3. while 循环（先判断再执行）

int i = 0;
while (i < 3) {
    System.out.println("while: " + i);
    i++;
}

🔹 4. do...while 循环（至少执行一次）

int i = 0;
do {
    System.out.println("do while: " + i);
    i++;
} while (i < 3);

✅ 四、跳转语句

关键字用法说明
break跳出当前循环或 switch
continue跳过当前循环的剩余部分，继续下一轮
return返回方法调用结果并结束当前方法

示例：

for (int i = 1; i <= 5; i++) {
    if (i == 3) {
        continue; // 跳过3
    }
    if (i == 5) {
        break; // 提前退出
    }
    System.out.println(i);
}

✅ 五、流程控制结构图（简略）

if-else        → 选择结构
switch-case    → 多分支结构
for / while    → 条件循环
do...while     → 保底循环（至少一次）
break/continue → 控制跳转
