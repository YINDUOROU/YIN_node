✅ Java 字符串（String）详解

这是 Java 开发中最常见、最核心的类型之一，面试必考 + 项目必用 + 细节最多

一、String 是什么？

String 是 Java 中用于表示一串字符的类，是引用类型，定义在 java.lang.String 包下。

⚠️ String 是 不可变（immutable）对象，一旦创建内容不能修改。

二、String 的常用创建方式

String s1 = "Hello";                     // 字面量方式（常量池）
String s2 = new String("Hello");         // 创建新对象（堆内存）

📌 字符串常量池机制（面试重点）：

  • "Hello" 字面量存放在字符串常量池中，如果已存在，则重用。
  • new String("Hello") 每次都在堆中新建对象，即使内容相同。
String a = "abc";
String b = "abc";
System.out.println(a == b); // true：指向常量池同一地址

String c = new String("abc");
System.out.println(a == c); // false：一个在常量池，一个在堆

✅ 三、String 的常用方法（超常用！）

String str = " Hello World ";
方法说明示例
length()获取长度str.length() → 13
charAt(i)获取第 i 个字符str.charAt(0) → ' '
substring(a, b)截取子串str.substring(1, 5) → "Hell"
toUpperCase()转大写"abc".toUpperCase() → "ABC"
trim()去除前后空格" abc ".trim() → "abc"
equals()判断内容是否相等"abc".equals("abc")
equalsIgnoreCase()忽略大小写比较"A".equalsIgnoreCase("a")
indexOf()查找子串位置"hello".indexOf("l") → 2
replace()替换内容"abc".replace("a", "x") → "xbc"
split()按指定符号切割字符串"a,b,c".split(",")

🔹 示例：

String name = "Alice";
System.out.println(name.length());          // 5
System.out.println(name.charAt(0));         // A
System.out.println(name.substring(1, 3));   // li
System.out.println(name.contains("li"));   // true
System.out.println(name.toUpperCase());     // ALICE

✅ 四、字符串拼接问题（性能陷阱）

🔸 错误方式：

String result = "";
for (int i = 0; i < 1000; i++) {
    result += i; // ❌ 每次拼接都创建新对象，极慢
}

🔸 推荐方式：用 StringBuilder

StringBuilder sb = new StringBuilder();
for (int i = 0; i < 1000; i++) {
    sb.append(i);
}
String result = sb.toString();

✅ 六、字符串比较的常见陷阱

String a = "abc";
String b = new String("abc");

System.out.println(a == b);         // false 比较的是地址
System.out.println(a.equals(b));    // true 比较的是内容

📌 总结：

  • == 比较地址（引用是否一样）
  • .equals() 比较内容（值是否一样）

✅ Java 的 Lambda 表达式（Lambda Expression）

Lambda 是 Java 8 引入的一种函数式编程风格，可以让你写出更简洁、优雅的代码，尤其在处理集合、回调、事件监听等场景中非常强大。

一、什么是 Lambda 表达式？

Lambda 表达式本质上是匿名函数，用于简洁地表示只有一个方法的接口实现（函数式接口）。

🔹 常规写法（Java 8 之前）：

Runnable r = new Runnable() {
    public void run() {
        System.out.println("Hello");
    }
};

🔹 使用 Lambda：

Runnable r = () -> System.out.println("Hello");

是不是一下清爽很多！

✅ 二、Lambda 表达式的语法格式

(参数列表) -> { 方法体 }

📌 常见形式对比：

写法示例
无参无返回() -> System.out.println("Hi")
有参无返回(x, y) -> System.out.println(x + y)
有参有返回(x, y) -> x + y
单个参数简写x -> x * 2
方法体多行需加花括号(x, y) -> { int z = x + y; return z; }

✅ 三、Lambda 使用的前提：函数式接口

🔹 什么是函数式接口？

只有一个抽象方法的接口，Java 8 提供了 @FunctionalInterface 注解来标识。
@FunctionalInterface
interface Calculator {
    int calc(int a, int b);
}


Calculator c = (a, b) -> a + b;
System.out.println(c.calc(3, 4)); // 7

✅ 四、常见的函数式接口（Java 已内置）

接口名方法说明
Runnablevoid run()无参无返回
Callable<T>T call()有返回值
Consumer<T>void accept(T t)接收参数，无返回（forEach）
Supplier<T>T get()无参，有返回
Function<T, R>R apply(T t)一参一返回
Predicate<T>boolean test(T t)判断型接口

✅ 五、结合集合使用（超强写法）

List<String> names = Arrays.asList("Tom", "Jerry", "Lucy");

// 传统写法
for (String name : names) {
    System.out.println(name);
}

// Lambda 写法
names.forEach(name -> System.out.println(name));

// 方法引用（进阶）
names.forEach(System.out::println);

✅ 六、结合 Stream 流式编程使用（高阶）

List<String> names = Arrays.asList("Tom", "Jerry", "Lucy");

names.stream()
     .filter(name -> name.length() <= 3)   // 过滤
     .map(String::toUpperCase)             // 映射
     .sorted()
     .forEach(System.out::println);        // 打印

七、Lambda 的本质是什么？

Lambda 是 语法糖，Java 编译器会把它转换为：

  1. 匿名内部类
  1. 或者字节码中的 invokedynamic 指令（更底层）

所以它的本质是 简化写法 + 保持类型安全
