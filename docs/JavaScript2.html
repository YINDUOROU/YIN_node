JavaScript 函数定义与调用

函数是 JavaScript 的核心机制之一，它是一段可以重复执行的代码块，用于组织逻辑、减少冗余、提高可维护性。

一、函数的定义方式

1. 使用 function 关键字声明（命名函数）

javascript

function greet(name) {
  return "Hello, " + name;
}

2. 匿名函数表达式（函数作为值赋给变量）

javascript

const greet = function(name) {
  return "Hi, " + name;
};

3. 箭头函数（ES6）

javascript

const greet = (name) => {
  return "Welcome, " + name;
};
  • 如果只有一个参数、一个返回值可简写为：
javascript

const greet = name => "Hello, " + name;

二、函数调用

函数必须调用才会执行：

javascript

let result = greet("Tom");
console.log(result); // 输出 Hello, Tom

三、函数参数

函数可以定义多个参数，用逗号分隔：

javascript

function sum(a, b) {
  return a + b;
}
console.log(sum(3, 4)); // 输出 7

四、默认参数值（ES6）

如果调用时没有提供参数，可以为其设置默认值：

javascript

function greet(name = "游客") {
  return "你好，" + name;
}
console.log(greet()); // 输出 你好，游客

五、返回值 return

函数可以使用 return 返回一个值，并立即结束函数执行。

javascript

function multiply(a, b) {
  return a * b;
  console.log("不会执行"); // return 后的代码不会执行
}

六、函数作为参数（高阶函数）

函数可以作为参数传递给另一个函数：

javascript

function sayHi() {
  return "Hi";
}

function callFunction(fn) {
  console.log(fn()); // 执行传入的函数
}

callFunction(sayHi); // 输出 Hi

七、函数作为返回值

javascript

function outer() {
  return function() {
    console.log("我是内部函数");
  };
}

const fn = outer();
fn(); // 输出：我是内部函数

八、立即执行函数（IIFE）

函数可以定义后立刻执行，常用于模块封装：

javascript

(function() {
  console.log("立即执行！");
})();

或 ES6 箭头函数：

javascript

(() => {
  console.log("立即执行！");
})();

九、函数总结

项目说明
命名函数可重复调用，便于调试
匿名函数常用于回调、作为参数传递
箭头函数简洁写法，this 指向外层作用域
IIFE封装作用域，常用于模块/立即执行逻辑

JavaScript 事件绑定与处理（事件流）

在 Web 编程中，JavaScript 常用于响应用户的交互操作，例如点击、键盘输入、鼠标移动等，这就是所谓的“事件处理”。

一、常见事件类型

事件名称说明
onclick鼠标点击
onmouseover鼠标移入
onmouseout鼠标移出
onkeydown按下键盘
onload页面或资源加载完成
oninput表单输入变更
onsubmit表单提交

二、绑定事件的方式

1. HTML 方式（不推荐）

html

<button onclick="sayHello()">点击我</button>

2. DOM 元素绑定（推荐）

javascript

const btn = document.getElementById("myBtn");
btn.onclick = function() {
  alert("按钮被点击");
};

3. addEventListener（推荐，支持多个事件）

javascript

btn.addEventListener("click", function() {
  alert("点击事件被触发");
});

三、事件对象 event

在事件触发时，会传入一个事件对象 event，包含当前事件的各种信息。

javascript

btn.addEventListener("click", function(e) {
  console.log(e.target); // 触发事件的元素
});

四、事件移除

可以使用 removeEventListener 移除绑定的事件。

javascript

function sayHi() {
  alert("Hi!");
}
btn.addEventListener("click", sayHi);

// 移除
btn.removeEventListener("click", sayHi);

五、事件流（捕获阶段 → 目标阶段 → 冒泡阶段）

事件从根节点向目标元素传递（捕获），再从目标返回（冒泡）：

javascript

element.addEventListener("click", handler, true);  // 捕获
element.addEventListener("click", handler, false); // 冒泡（默认）

六、阻止事件冒泡

javascript

event.stopPropagation();

七、阻止默认行为

比如阻止表单提交、链接跳转：

javascript

event.preventDefault();

八、事件绑定小结

方法特点
HTML 事件（内联）简单，但侵入性高，不推荐
element.onclick覆盖旧事件，不支持多个绑定
addEventListener标准推荐，可绑定多个函数
removeEventListener移除指定事件处理器

九、事件示例：按钮点击

html

<button id="btn">点击我</button>

<script>
  document.getElementById("btn").addEventListener("click", function() {
    alert("你点击了按钮！");
  });
</script>

JavaScript 作用域与作用域链（Scope & Scope Chain）

“作用域”是变量的生存空间，是理解 JavaScript 的核心基础之一，决定了变量在何处可访问，函数在何处被调用。

一、什么是作用域（Scope）

作用域： 指程序中定义变量的区域及其可访问性。

JavaScript 中的作用域包括：

  • 全局作用域（Global Scope）
  • 函数作用域（Function Scope）
  • 块级作用域（Block Scope，ES6 引入）

二、全局作用域

在任何函数或块外部声明的变量，属于全局作用域，可被整个脚本访问。

javascript

let a = 100; // 全局变量

function show() {
  console.log(a); // 100，可访问
}

📌 在浏览器中，全局变量会绑定到 window 对象：

javascript

var b = 200;
console.log(window.b); // 200

三、函数作用域（Function Scope）

函数内部声明的变量，只能在该函数内部访问。

javascript

function test() {
  let msg = "Hello";
  console.log(msg); // ✅ 有效
}
console.log(msg); // ❌ 报错：msg is not defined
📌 JavaScript 中的 var 是函数作用域，而 let / const 是块级作用域（见下文）。

四、块级作用域（Block Scope）【ES6 新增】

使用 let 或 const 声明的变量，在 {} 块中有效。

javascript

{
  let a = 10;
  const b = 20;
}
console.log(a); // ❌ 报错

⚠️ var 没有块级作用域：

javascript

{
  var x = 5;
}
console.log(x); // ✅ 输出 5

五、作用域链（Scope Chain）

当访问一个变量时，JavaScript 引擎会从当前作用域向外层作用域逐级查找，直到全局作用域为止。

javascript

let x = 1;

function outer() {
  let y = 2;

  function inner() {
    let z = 3;
    console.log(x, y, z); // 1, 2, 3
  }

  inner();
}

outer();
📌 如果在作用域链中都找不到变量，则会抛出 ReferenceError。

六、词法作用域（静态作用域）

JavaScript 是词法作用域语言，作用域在定义时决定，与运行时无关。

javascript

let a = "全局";

function test() {
  console.log(a);
}

function run() {
  let a = "局部";
  test(); // 输出：全局，而不是“局部”
}

run();
❗这与“动态作用域”的语言（如某些 shell 脚本语言）不同。

七、扩展：作用域与闭包的关系

闭包是“作用域链”的应用场景。

当一个函数访问了它外部函数的变量，并将其返回或传出时，就形成了闭包（Closure）。这使得局部变量得以保存在内存中。

javascript

function outer() {
  let count = 0;
  return function inner() {
    count++;
    console.log(count);
  };
}

const counter = outer();
counter(); // 1
counter(); // 2

🧠 你可以理解为：

  • 作用域链保证了 count 在外层函数结束后仍然可被访问；
  • 闭包延长了局部变量的生命周期，可能造成内存泄漏风险（需小心）。

八、最佳实践与注意事项

✅ 推荐使用 let / const 来声明变量，避免作用域提升和污染：

javascript

for (let i = 0; i < 3; i++) {
  setTimeout(() => console.log(i), 100); // 输出 0, 1, 2
}

❌ 若用 var 会输出 3, 3, 3，因为 var 是函数级作用域：

javascript
for (var i = 0; i < 3; i++) {
  setTimeout(() => console.log(i), 100); // 3, 3, 3
}

✅ 避免全局变量污染，封装变量：

javascript

(function() {
  const privateData = "隐藏变量";
  console.log(privateData);
})();

九、总结图解（思维导图建议）

作用域的结构：


复制编辑
全局作用域
 └── 函数作用域
      └── 块级作用域

查找变量时，向外层逐级查找 —— 这就是作用域链（Scope Chain）。

JavaScript 对象创建与访问（Object）

对象是 JavaScript 中最重要的数据类型之一，是一组“键值对（key-value）”的集合，用于描述一类事物的属性和行为。

一、什么是对象？

对象是由属性和方法组成的结构：

javascript

let person = {
  name: "Tom",              // 属性
  age: 20,
  sayHello: function() {    // 方法
    console.log("Hello!");
  }
};
📌 属性是值，方法是函数

二、创建对象的方式

1. 字面量创建法（推荐）

javascript

const car = {
  brand: "Toyota",
  year: 2022
};

✔️ 简洁直观，是最常用的方式。

2. 使用 new Object()

javascript

const book = new Object();
book.title = "JavaScript 入门";
book.price = 59;

⚠️ 实际开发中较少使用。

3. 使用构造函数

javascript

function Animal(name, age) {
  this.name = name;
  this.age = age;
}

const dog = new Animal("Lucky", 3);

✔️ 用于创建“多个相同结构”的对象。

4. 使用工厂函数（更灵活）

javascript

function createUser(name, email) {
  return {
    name,
    email,
    isAdmin: false
  };
}

const user1 = createUser("小明", "test@example.com");
✅ 推荐用于生成“定制对象”，灵活不依赖 new。

5. 使用 Object.create()（继承推荐）

javascript

const base = { greet() { console.log("Hi"); } };
const child = Object.create(base);
child.name = "Child";

✔️ 用于构建原型链，底层封装框架常用。

三、访问对象属性

有两种方式访问对象属性：

1. 点（.）语法

javascript

console.log(person.name); // Tom

2. 方括号（[]）语法

javascript

console.log(person["age"]); // 20
✅ 方括号方式适用于：
  • 属性名是动态变量时
  • 属性名包含空格、符号时
javascript

let key = "name";
console.log(person[key]); // Tom

const obj = {
  "first name": "Alice"
};
console.log(obj["first name"]);

四、修改对象属性

javascript

person.age = 25;
person["name"] = "Jerry";

五、添加/删除属性

javascript

person.city = "Tokyo";   // 添加
delete person.age;       // 删除

六、检测属性存在性

javascript

"name" in person         // true
person.hasOwnProperty("age")  // false（已被删）

七、对象遍历（for...in）

javascript

for (let key in person) {
  console.log(key + ": " + person[key]);
}

📌 只会遍历可枚举的自有/继承属性，可配合 hasOwnProperty 过滤：

javascript

for (let key in person) {
  if (person.hasOwnProperty(key)) {
    console.log(key);
  }
}

八、对象方法简写（ES6）

javascript

const student = {
  name: "Lily",
  sayHi() {
    console.log("你好！");
  }
};
student.sayHi(); // 你好！

九、对象扩展：动态属性名（ES6）

javascript

let key = "level";
let user = {
  name: "Bob",
  [key]: "Admin" // 等同于 user["level"] = "Admin"
};

🔍 拓展：对象的深拷贝与浅拷贝

javascript

const obj1 = { a: 1, b: { c: 2 } };
const obj2 = Object.assign({}, obj1); // 浅拷贝
const obj3 = structuredClone(obj1);   // ✅ 深拷贝（现代浏览器支持）

操作方法示例
创建对象字面量 / 构造函数 / 工厂函数
属性访问obj.key or obj["key"]
属性增删改查obj.key = ... / delete obj.key
遍历对象for...in / Object.keys()
动态属性名[变量名] 形式

JavaScript 数组（Array）对象与操作方法

数组是用于存储一组有序数据的对象。JavaScript 数组功能强大，是开发中使用频率极高的数据结构。

一、创建数组的方式

1. 字面量（推荐方式）


const fruits = ["apple", "banana", "orange"];

2. 使用 new Array()（不推荐）

javascript

const nums = new Array(1, 2, 3);      // ✅ 正确
const empty = new Array(5);          // ❌ 创建长度为5的稀疏数组

✔️ 字面量更安全、直观。

二、访问数组元素

数组下标从 0 开始：

javascript

console.log(fruits[0]); // apple
console.log(fruits[2]); // orange

三、修改/添加/删除元素

修改元素：

javascript

fruits[1] = "pear";

添加元素：

javascript

fruits[3] = "mango";
fruits.push("kiwi");         // 添加到末尾
fruits.unshift("grape");     // 添加到开头

删除元素：

javascript

fruits.pop();     // 删除末尾元素
fruits.shift();   // 删除开头元素
delete fruits[1]; // ❌ 留空位，不建议

四、数组常用属性与方法

1. .length 长度属性

javascript

console.log(fruits.length); // 数组元素个数

2. .push() / .pop()

javascript

arr.push("元素")     // 末尾添加
arr.pop()           // 删除末尾

3. .unshift() / .shift()

javascript

arr.unshift("头")    // 开头添加
arr.shift()         // 删除开头

4. .indexOf() / .includes()

javascript

arr.indexOf("apple")    // 找到返回索引，否则 -1
arr.includes("banana")  // 返回 true / false

五、数组遍历方法

1. for 经典循环

javascript

for (let i = 0; i < arr.length; i++) {
  console.log(arr[i]);
}

2. for...of（推荐）

javascript

for (const item of arr) {
  console.log(item);
}

3. forEach() 高阶函数

javascript

arr.forEach((item, index) => {
  console.log(index + " = " + item);
});

六、数组高级操作方法

1. map() → 映射转换新数组

javascript

const squares = [1, 2, 3].map(n => n * n); // [1, 4, 9]

2. filter() → 筛选符合条件的元素

javascript

const even = [1, 2, 3, 4].filter(n => n % 2 === 0); // [2, 4]

3. reduce() → 聚合计算

javascript

const sum = [1, 2, 3].reduce((acc, cur) => acc + cur, 0); // 6

4. find() / findIndex()

javascript

arr.find(item => item.length > 5);       // 返回匹配项
arr.findIndex(item => item === "abc");   // 返回索引

5. sort() → 排序（注意必须传比较函数）

javascript

const arr = [4, 2, 10];
arr.sort((a, b) => a - b); // 升序 [2, 4, 10]

6. slice() / splice()

javascript

arr.slice(1, 3);  // 截取（不改变原数组）
arr.splice(1, 2); // 删除或替换元素（改变原数组）

七、数组与字符串互转

javascript

const str = arr.join(", "); // 数组 → 字符串
const newArr = str.split(", "); // 字符串 → 数组

八、数组判定方式

javascript

Array.isArray(arr);     // ✅ 推荐
typeof arr;             // ❌ 返回 "object"，不可靠

九、二维数组（矩阵）

javascript

const matrix = [
  [1, 2],
  [3, 4]
];

console.log(matrix[1][0]); // 3

🔍 补充：解构与展开运算符（ES6+）

javascript

const [first, ...rest] = [10, 20, 30]; // 解构赋值
const merged = [...arr1, ...arr2];     // 展开合并

JavaScript Date 日期对象与时间处理

Date 是 JavaScript 用于表示时间的内建对象，包含年、月、日、时、分、秒、毫秒等信息，并支持获取、设置、格式化、计算等操作。

一、创建日期对象的方式

1. 当前时间

javascript

const now = new Date();
console.log(now); // 当前日期时间

2. 指定时间字符串

javascript

const d1 = new Date("2024-05-23 18:30:00");
⚠️ 建议使用 YYYY-MM-DD 这种标准格式，避免地区兼容问题。

3. 年/月/日/时/分/秒/毫秒参数

javascript

const d2 = new Date(2024, 4, 23, 18, 30, 0); // 月份从 0 开始

4. 时间戳（毫秒数）

javascript

const d3 = new Date(1716450000000);

二、获取时间相关信息

方法名含义示例返回值（假设 2024-05-23）
.getFullYear()获取完整年份2024
.getMonth()获取月份（0~11）4
.getDate()获取日（1~31）23
.getDay()获取星期（0=周日）4（周四）
.getHours()小时（0~23）18
.getMinutes()分钟（0~59）30
.getSeconds()秒数（0~59）0
.getMilliseconds()毫秒（0~999）0
.getTime()获取时间戳（ms）1716450000000

示例：

javascript

const date = new Date();
console.log(date.getFullYear(), date.getMonth() + 1, date.getDate());

三、设置时间

与 getXxx() 方法对应，也有 setXxx() 方法：

javascript

date.setFullYear(2030);
date.setMonth(0); // 一月
date.setDate(1);  // 设置为 1 号

四、时间戳转换

当前时间戳（1970 年以来的毫秒数）

javascript

Date.now();             // 推荐方式
new Date().getTime();   // 旧写法

时间戳转日期

javascript

const ts = 1716450000000;
const date = new Date(ts);

五、时间格式化

JavaScript 原生不支持复杂格式化，但常见写法如下：

javascript

function formatDate(date) {
  const y = date.getFullYear();
  const m = String(date.getMonth() + 1).padStart(2, "0");
  const d = String(date.getDate()).padStart(2, "0");
  const h = String(date.getHours()).padStart(2, "0");
  const min = String(date.getMinutes()).padStart(2, "0");
  const s = String(date.getSeconds()).padStart(2, "0");
  return `${y}-${m}-${d} ${h}:${min}:${s}`;
}

console.log(formatDate(new Date()));

六、时间计算与比较

1. 加减毫秒（如加一天）

javascript

const tomorrow = new Date(Date.now() + 24 * 60 * 60 * 1000);

2. 两个日期之间相差毫秒数

javascript

const start = new Date("2024-01-01");
const end = new Date("2024-05-23");
const diff = end - start; // 单位：毫秒
console.log(diff / (1000 * 60 * 60 * 24)); // 转换成天数

七、常用日期库推荐（强烈建议）

由于原生 Date 不支持格式化、时区、本地化，建议使用以下库：

库名特点说明
dayjs🔥 轻量小巧、语法优雅（推荐）
date-fns函数式 API，按需引入，无全局污染
moment功能强大但较大，已被社区淘汰

示例（dayjs）：

javascript

import dayjs from "dayjs";

const now = dayjs();
console.log(now.format("YYYY-MM-DD HH:mm:ss")); // 格式化
console.log(now.add(7, "day").format());        // 加7天

八、面试 & 实战小技巧

✅ 判断是否为有效日期：

javascript

function isValidDate(d) {
  return d instanceof Date && !isNaN(d);
}

✅ 快速获取今天凌晨、昨天、明天的时间：

javascript

function getTodayZero() {
  const now = new Date();
  return new Date(now.getFullYear(), now.getMonth(), now.getDate());
}

JavaScript Math 对象（数学常用方法与技巧）

Math 是 JavaScript 的内建对象，用于执行各种数学计算。它不是构造函数，不能使用 new Math()，所有方法均以静态方式调用。

一、常用数学属性（常量）

二、基本数学方法

1. 四舍五入相关

javascript

Math.round(4.7);  // 四舍五入 → 5
Math.floor(4.7);  // 向下取整 → 4
Math.ceil(4.1);   // 向上取整 → 5
Math.trunc(4.9);  // 去除小数部分（不四舍五入）→ 4（ES6）
✅ trunc() 仅保留整数部分，适用于正负数统一处理。

2. 随机数（重点）

javascript

Math.random(); // 0 ≤ x < 1 之间的随机小数

生成任意范围内的整数：

javascript

// [min, max] 之间的随机整数
function randomInt(min, max) {
  return Math.floor(Math.random() * (max - min + 1)) + min;
}

3. 最大值 / 最小值

javascript

Math.max(3, 6, 2);  // 6
Math.min(3, 6, 2);  // 2

⚠️ 不支持数组直接传入（需配合扩展运算符）：

javascript

let arr = [1, 3, 2];
Math.max(...arr);  // ✅ 正确

4. 幂与根

javascript

Math.pow(2, 3);   // 2 的 3 次方 → 8
Math.sqrt(16);    // 平方根 → 4
Math.cbrt(27);    // 立方根（ES6）→ 3

5. 绝对值

javascript

Math.abs(-7);     // → 7

6. 三角函数（角度需转弧度）

javascript

Math.sin(Math.PI / 2); // 1
Math.cos(0);           // 1
Math.tan(Math.PI / 4); // ≈ 1

角度转弧度函数：

javascript

function toRadians(deg) {
  return deg * Math.PI / 180;
}

7. 对数与指数

javascript

Math.log(10);      // 以 e 为底
Math.log10(100);   // 以 10 为底（ES6）
Math.exp(1);       // e 的 1 次方

三、实战技巧场景

✅ 保留指定位小数（非字符串）

javascript
复制编辑
function roundTo(n, digits) {
  const factor = Math.pow(10, digits);
  return Math.round(n * factor) / factor;
}

roundTo(3.14159, 2); // → 3.14

✅ 数组最大/最小值

javascript

const scores = [88, 99, 74, 100];
const max = Math.max(...scores);
const min = Math.min(...scores)

✅ 自定义取整函数

javascript

// 向下取整到 10 的倍数
function floorTo10(n) {
  return Math.floor(n / 10) * 10;
}

JavaScript 正则表达式 RegExp（字符串模式匹配神器）

正则表达式（Regular Expression）是用于匹配、查找、替换字符串中字符模式的强大工具，在表单验证、文本提取、数据清洗等场景中广泛使用。

一、创建正则表达式的方式

1. 字面量形式（推荐）

javascript

const regex = /abc/;

2. 构造函数形式（可动态传参）

javascript

const regex = new RegExp("abc");

二、正则表达式常用元字符

元字符含义示例
.匹配任意单个字符（换行符除外）/a.b/ → ab, acb
\d匹配任意一个数字（[0-9]）/\d\d\d/ → 123
\w匹配字母/数字/下划线（[A-Za-z0-9_]）/\w\w/
\s匹配任意空白字符/\s+/
\b匹配单词边界/\bcat\b/
^匹配输入开始/^a/
$匹配输入结束/end$/
[]字符集，匹配其中任意一个字符/[abc]/
[^]否定字符集，排除其中字符/[^abc]/
``或
()分组/(abc)+/

三、量词（重复匹配）

量词含义示例
*匹配 0 次或多次/a*/
+匹配 1 次或多次/a+/
?匹配 0 次或 1 次/a?/
{n}精确匹配 n 次/a{3}/
{n,}至少匹配 n 次/a{2,}/
{n,m}匹配 n 到 m 次/a{2,4}/

四、常用正则匹配案例

1. 验证手机号（11位中国大陆）

javascript

/^1[3-9]\d{9}$/

2. 验证邮箱

javascript

/^[\w.-]+@[a-zA-Z0-9.-]+\.[a-zA-Z]{2,}$/

3. 验证身份证（简单版）

javascript

/^\d{15}$|^\d{17}[\dXx]$/

4. 匹配中文字符

javascript

/[\u4e00-\u9fa5]/g

5. 去除字符串空格

javascript

str.replace(/\s+/g, "");

五、RegExp 对象常用方法

1. .test()：是否匹配

javascript

/abc/.test("abcdef"); // true

2. .exec()：返回匹配结果对象（可提取组）

javascript

const result = /(\d+)-(\d+)/.exec("2024-05");
console.log(result[1]); // 2024

六、字符串方法配合正则使用

1. match()

javascript

"abc123".match(/\d+/); // → ["123"]

2. replace()

javascript

"abc123".replace(/\d+/, "***"); // → abc***

支持函数作为替换：

javascript

"price:99".replace(/\d+/, m => Number(m) + 1); // price:100

3. search()

javascript

"abc123".search(/\d+/); // → 索引位置

4. split()

javascript

"a,b c".split(/[\s,]+/); // → ["a", "b", "c"]

七、修饰符（Flags）

修饰符含义
g全局匹配
i忽略大小写
m多行匹配（影响 ^ $）
javascript

const regex = /abc/gi;

八、分组与反向引用

1. 分组提取

javascript

const date = "2024-05-23";
const match = /(\d{4})-(\d{2})-(\d{2})/.exec(date);
console.log(match[1], match[2], match[3]); // 2024 05 23

2. 分组替换

javascript

"2024-05-23".replace(/(\d{4})-(\d{2})-(\d{2})/, "$2/$3/$1"); // 05/23/2024

九、零宽断言（高级）

正向预查 (?=pattern)

javascript

/\d+(?=元)/.exec("价格为99元"); // → 99

负向预查 (?!pattern)

javascript

/\d+(?!元)/.exec("共100张票"); // → 100

🔍 实战建议与陷阱

  • .* 是贪婪匹配，尽量使用 .*? 避免吃太多字符
  • 特殊字符（如 .、+、）要加 \ 转义
  • 验证复杂格式（如 URL、HTML）推荐用正则库或 DOM 解析
